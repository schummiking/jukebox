# Jukebox Public

QQ 音乐 FLAC/MP3 下载器。单个 HTML 文件，零安装零依赖，用自己的 QQ 音乐账号。

## 使用

1. 双击 `jukebox.html` 打开
2. 首次使用需要登录授权（按页面提示操作，约 1 分钟）
3. 授权后搜歌、点下载

## 授权流程

1. 点击页面上的链接打开 y.qq.com 并登录
2. 复制页面提供的代码，粘贴到 y.qq.com 地址栏（删掉开头的 * 号后回车）
3. 将弹出的内容粘贴回工具页面，点保存

授权信息保存在浏览器本地，关闭后再打开仍然有效。Token 每次打开自动刷新。

## 要求

- QQ 音乐会员账号（绿钻/豪华绿钻）
- 现代浏览器（Chrome / Edge / Firefox）

## 安全

- 授权信息只存在你自己的浏览器里，不上传到任何服务器
- 所有请求直接从你的浏览器到 QQ 音乐，无中间人
- HTML 文件本身不包含任何人的账号信息，可以公开分享
